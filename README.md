# Vue纯净版开发环境(使用vue-cli@3创建)
## 配置详情:
- 使用cli初始化配置vuex vue-route less
- 手动配置axios,iview
- 使用express做后端接口(3000)完成
- 使用webpackDev完成代理跨域
****
## 使用说明
- 前提条件:<br>
    安装node环境,项目文件夹中`npm install`安装依赖
- service文件夹中`node app.js`, 开启后台接口
- 项目文件夹中`npm run serve`运行Vue项目
- 即可在浏览器**127.0.0.1:8080**打开项目
